In the spirit of Atomic Habits, I pick one new keyboard shortcut to practice each week. I use many of these daily:

## Google Suite

- `⌘⌥M` — Add a comment
- `⌘⇧↑` / `⌘⇧↓` — Foreground/background image (Slides)

## macOS Finder

- `⌘N` — New window
- `⌘⇧N` — New folder

## Firefox

- `⌘l` - Jump to nav bar

## VS Code

- `⌥` + click - Add a second cursor

## Many programs

- `⌘[` / `⌘]` - Indent up or down 1 level
